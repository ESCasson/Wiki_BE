# Wiki_BE
basic wiki with graphql, apollo express mongoDB
To install

1. master branch
1. create a mondoDB Compass
1. create a database called shorewikidb
1. within the database create two collections
   a. areas
   b. sectiontexts
1. run npm install
1. run node server.js

Backend tech

1. GraphQL
2. noSQL db
3. apollo
